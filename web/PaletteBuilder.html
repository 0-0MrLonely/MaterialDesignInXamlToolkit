Material Design in XAML Toolkit Palette Builder

This isn't a web demo/style resource for Material Design, but is a tool for building a Material Design palette and ResourceDictionary for XAML applications using the Material Design in XAML Toolkit.

Select a primary and accent colour from the swatches on the right. Then, in your palette select the primary huues, and a hue for the accent.
